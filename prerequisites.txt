Distributed File Storage System - Prerequisites

---
Backend (Go):
1. Go Language (v1.20+ recommended)
2. MySQL Server (v8.0+ recommended)
   - Must create a database named 'dsdb' (or as configured in .env).
3. Apache Zookeeper (for leader election/distributed coordination)
   - Running at localhost:2181 or configured IP in main.go.
4. Git (for cloning and version control)
5. Go Dependencies (automatically installed via 'go mod tidy'):
   - Gin (HTTP Framework)
   - GORM (ORM for MySQL)
   - NTP client
   - CompileDaemon (Optional, for hot reloading: 'go install github.com/githubnemo/CompileDaemon@latest')

Frontend (React + Vite + TypeScript):
1. Node.js (v18+ recommended)
2. NPM or Yarn
3. Vite (will be installed via project dependencies)

General:
1. Environment Variables:
   - PORT (default 8080)
   - DB_HOST, DB_PORT, DB_USER, DB_PASSWORD, DB_NAME
   - NODE_ID (required for the Go backend node to identify itself)
